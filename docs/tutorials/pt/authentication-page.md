Na página Autenticação, é possível escolher quais formas de login você deseja oferecer aos compradores e operadores da sua loja VTEX. Esta página está disponível em _Configurações da conta_ > _Autenticação_.

![Página de autenticação](//images.contentful.com/alneenqid6w5/6cbXmuu9GiTfuYnIsuFKQp/986b144c0704f780d774f8f3790ba8e6/Tela_de_Autentica____o_PT.png)

## Formas de login

Atualmente, há quatro formas disponíveis de fazer login:

* **Código de acesso: **um código numérico aleatório é enviado para o email do usuário e este utiliza o código para realizar o login.
* **Senha: **o usuário pode registrar uma senha e efetuar o login usando o email e a senha cadastrados.
* **Facebook: **o usuário faz login usando sua conta do Facebook.
* **Google: **o usuário faz login usando sua conta do Google.

| Forma de login   | Loja virtual        | Admin                  |
|------------------|---------------------|------------------------|
| Código de acesso | Pode ser habilitada | Está sempre disponível |
| Senha            | Pode ser habilitada | Está sempre disponível |
| Facebook         | Pode ser habilitada | Não está disponível    |
| Google           | Pode ser habilitada | É selecionável         |

<div class="alert alert-info">
Pelo menos uma das quatro formas de login deve estar habilitada para a loja virtual
</div>

## Loja virtual

Na aba **Loja virtual** você pode escolher quais modos os compradores podem utilizar para fazer login na sua loja. Para realizar a configuração para habilitar o login via Facebook ou via Google é necessário clicar no respectivo botão **Editar**. 

Para mais informações sobre como realizar a configuração, verifique o artigo [Configurar login com Facebook e Google](https://help.vtex.com/pt/tutorial/integracao-google-e-facebook-para-login--tutorials_513).

## Admin

Na aba **Admin** você pode escolher se operadores podem efetuar login através da conta do Google. Para realizar a configuração para habilitar o login via Google é necessário clicar no botão **Editar**. 

Para mais informações sobre como realizar a configuração, verifique o artigo [Cadastrar Client Id e Client Secret para login com Google](https://help.vtex.com/pt/tutorial/cadastrar-client-id-e-client-secret-para-login-com-google).
