O YamiSplitMoipV1 é um gateway dedicado que extende funcionalidades do Moip. Com ele, é possível realizar splits de pagamento nos marketplaces (solução para o usuário fazer um único pagamento, mesmo comprando produtos de sellers diferentes) usando o subadquirente.

## Configurar afiliação de gateway YamiSplitMoipV1
1. No Admin VTEX, acesse **Configurações da loja** > **Pagamentos** > **Configurações**, ou digite **Configurações** na barra de busca no topo da página.
2. Na aba __Afiliações de Gateways__, clique no botão __+__.
3. Clique no conector __YamiSplitMoipV1__.
4. Preencha os campos _Application Key_ e _Application Token_ com as informações da sua conta __Moip__.
5. Clique em __Salvar__.

## Configurar condição de pagamento

Depois de seguir os passos indicados, o YamiSplitMoipV1 vai estar configurado na sua loja. Assim, quando você for cadastrar uma condição de pagamento para ser usada em um marketplace, ele vai estar disponível no campo __Processar com a afiliação__. 

Para saber como definir condições de pagamento, acesse [este artigo do Help](/pt/tutorial/condicoes-de-pagamento).
