## Sumário

<div class="alert alert-info">
  <p>Este problema conhecido foi traduzido automaticamente do inglês.</p>
</div>


A IU não mostra o formulário correto para editar o endereço selecionado na etapa de Envio ao selecionar para adicionar um novo endereço primeiro.


##

## Simulação



- Ter um usuário com um perfil completo;
- Clique para adicionar um novo endereço, não digite/selecione nada e volte para a lista de endereços;
- Clique para editar o endereço selecionado;
- Você verá a mesma forma como ao adicionar um novo endereço.


##

## Workaround


Recarregue a página para poder editar novamente o endereço selecionado.
