O status das transações é diferente uma vez que você as abre em comparação com o que é mostrado na lista da transação.



## Simulação


Como isso não acontece em todas as transações, a maneira de simular é entrar em uma transação aleatória e verificar se o status dentro dela é o mesmo que está definido na lista de transações.
Uma vez que se entra na transação, de fato, o status é diferente
