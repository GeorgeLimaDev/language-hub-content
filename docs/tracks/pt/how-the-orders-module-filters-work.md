O módulo **Pedidos** oferece uma série de filtros diferentes, para facilitar a consulta e especificar buscas, de acordo com os critérios desejados.

Para utilizar um filtro, siga os passos abaixo:

1. No Admin VTEX, acesse **Pedidos > Todos os pedidos**, ou digite **Todos os pedidos** na barra de busca no topo da página.
2. Clique em `Filtros`.
3. Selecione o [filtro](https://help.vtex.com/pt/tutorial/filtrar-todos-pedidos--tutorials_192) desejado.
4. Clique em `Aplicar`.

Para saber mais detalhes sobre os filtros, veja o artigo [Filtrar pedidos em Todos os pedidos](https://help.vtex.com/pt/tutorial/filtrar-todos-pedidos--tutorials_192).
